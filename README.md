이진열 스토리(비밀 연구소)

게임은 턴수 베이스

장애물은 적 -

초급 - 장애물 없음

중급 - 움직이는 적

고급 - 레이저 추가

아이템 - 시간 정지, 대쉬, 추가 생명(보호막), 레이저 멈추는 거, 턴 수 늘려주는 아이템, 목표 지점 힌트

벌칙 - 적 추가(초급에서는 X), 이동 횟수 감소, 시야 감소, 레이저 빈도 증가, 아이템 사라지게

준혁 - 기본 플레이 구현

준우 - 적 멈춤, 여러 칸 이동

진열 - 보호막, 목표지점 힌트

명근 - 턴 수 늘려주는 거, 적 개수

변수명은 Lower camel case(ex: lowerCamelCase), 함수명과 은 snake case(ex: snake_case) 로 통일해주세요!
